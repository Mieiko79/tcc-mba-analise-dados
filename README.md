# 📊 TCC - MBA em Análise de Dados - USP/ESALQ

## 📌 Descrição
Este projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC) do MBA em Análise de Dados.

O objetivo é realizar a análise de dados aplicada à identificação de sentimentos em depoimentos de clientes, com foco em contextos de banco comercial e banco digital.

---

## 🎯 Objetivo
- Analisar dados textuais de clientes
- Identificar padrões de sentimentos
- Gerar insights para apoio à tomada de decisão

---

## 🛠️ Tecnologias Utilizadas
- Python
- Pandas
- Numpy
- (adicione aqui se usar: Power BI, Excel, SQL, etc.)

---

## 🔄 Etapas do Projeto
1. Coleta de dados  
2. Tratamento e limpeza  
3. Análise exploratória  
4. Processamento de linguagem natural (NLP)  
5. Geração de resultados e insights  

---

## 📈 Resultados Esperados
- Classificação de sentimentos (positivo, negativo, neutro)
- Identificação de padrões em feedbacks de clientes
- Apoio na tomada de decisão estratégica

## 📎 Observações
Este projeto tem caráter acadêmico e foi desenvolvido como parte do MBA em Análise de Dados.
